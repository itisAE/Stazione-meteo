ITIS Mario Delpozzo Weather Station

An educational project with artificial intelligence for weather forecasting

Real-time weather monitoring and forecasting for the city of Cuneo

Our Project

The ITIS Weather Station is an advanced weather monitoring and forecasting system designed and built by four students from the Robotics program at ITIS Mario Delpozzo in Cuneo.

The project combines weather monitoring hardware with machine learning techniques to offer not only real-time data, but also reliable forecasts for the next three days. Using historical data provided by ARPA Piemonte, our artificial intelligence system is able to predict weather conditions with a reasonable level of accuracy.

Project Objectives

  • Create a functional and reliable weather station
  • Collect real-time weather data
  • Develop an automated data archiving system
  • Implement a web interface for data visualization
  • Create a searchable historical archive
  • Develop a predictive model based on machine learning
  • Provide reliable three-day weather forecasts
  • Provide a useful service to the school and local community

Weather Forecasting System with Machine Learning

The main innovation of our project is the implementation of an artificial intelligence system for weather forecasting.

Using linear regression techniques, our model analyzes historical weather data provided by ARPA Piemonte for the Cuneo area and identifies patterns and correlations that allow us to predict weather conditions for the next three days.

Machine Learning Pipeline

Data Collection

Acquiring historical data from ARPA and real-time data from our sensors

Preprocessing

Cleaning, normalizing and preparing data for training

Training

Linear regression on historical data to identify weather patterns

Forecasting

Generating weather forecasts for the next three days
